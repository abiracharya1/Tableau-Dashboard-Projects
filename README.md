# Tableau-Dashboard-Projects
Business data visualization project using Tableau to analyze sales trends, customer performance and regional metrics.


Tableau Sales Dashboard
Project Overview

This project presents an interactive Tableau dashboard analyzing sales performance, regional contribution, and product trends. The dashboard enables quick business insights through visual exploration.

Business Problem

Organizations require visual reporting to understand sales performance and identify trends. Raw data makes it difficult to identify patterns without visualization.

This project demonstrates how Tableau dashboards support data driven decision making.
